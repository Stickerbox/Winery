# Half-Star Ratings Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Allow wines to be rated in 0.5 increments (1.0–5.0) instead of whole integers only.

**Architecture:** Three isolated changes — schema migration (Int → Float), action parsing (parseInt → parseFloat), and a full rewrite of the RatingStar component to use split hit zones and a half-fill overlay technique.

**Tech Stack:** Prisma + SQLite, Next.js 16 App Router, React 19, Tailwind CSS v4, lucide-react

### Task 3: Rewrite RatingStar with half-star support

**Files:**
- Modify: `components/ui/RatingStar.tsx`

**Approach:** Each star position renders:
1. A background (empty) `Star` SVG
2. A filled `Star` SVG clipped to 50% width (half) or 100% width (full) via an `overflow-hidden` wrapper
3. Two invisible absolutely-positioned `<button>` elements covering the left and right halves — left triggers `index + 0.5`, right triggers `index + 1.0`

On desktop, `onMouseEnter`/`onMouseLeave` preview values. On mobile, `onClick` commits them directly.

- [ ] **Step 1: Replace the full file contents**

Replace the entire contents of `components/ui/RatingStar.tsx` with:

```tsx
"use client";

import * as React from "react";
import { Star } from "lucide-react";
import { cn } from "@/lib/utils";

interface RatingStarProps {
    rating: number;
    maxRating?: number;
    onRatingChange?: (rating: number) => void;
    readonly?: boolean;
    className?: string;
}

export function RatingStar({
    rating,
    maxRating = 5,
    onRatingChange,
    readonly = false,
    className,
}: RatingStarProps) {
    const [hoverRating, setHoverRating] = React.useState<number | null>(null);
    const displayRating = hoverRating ?? rating;

    return (
        <div className={cn("flex items-center gap-1", className)}>
            {Array.from({ length: maxRating }).map((_, index) => {
                const halfValue = index + 0.5;
                const fullValue = index + 1;
                const fillAmount =
                    displayRating >= fullValue ? "full" :
                    displayRating >= halfValue ? "half" :
                    "empty";

                return (
                    <div key={index} className="relative h-6 w-6">
                        {/* Background empty star */}
                        <Star className="h-6 w-6 fill-transparent text-zinc-300 dark:text-zinc-600" />
                        {/* Filled overlay — clipped to left 50% for half, full width for full */}
                        {fillAmount !== "empty" && (
                            <div
                                className="absolute inset-0 overflow-hidden"
                                style={{ width: fillAmount === "half" ? "50%" : "100%" }}
                            >
                                <Star className="h-6 w-6 fill-yellow-400 text-yellow-400" />
                            </div>
                        )}
                        {/* Invisible hit zones (interactive mode only) */}
                        {!readonly && (
                            <>
                                <button
                                    type="button"
                                    className="absolute inset-y-0 left-0 w-1/2 cursor-pointer focus:outline-none"
                                    onMouseEnter={() => setHoverRating(halfValue)}
                                    onMouseLeave={() => setHoverRating(null)}
                                    onClick={() => onRatingChange?.(halfValue)}
                                />
                                <button
                                    type="button"
                                    className="absolute inset-y-0 right-0 w-1/2 cursor-pointer focus:outline-none"
                                    onMouseEnter={() => setHoverRating(fullValue)}
                                    onMouseLeave={() => setHoverRating(null)}
                                    onClick={() => onRatingChange?.(fullValue)}
                                />
                            </>
                        )}
                    </div>
                );
            })}
        </div>
    );
}
```
